## Исследование факторов, влияющих на успеваемость студентов

### Тема работы
Исследование факторов, влияющих на академическую успеваемость студентов на основе анализа анкетных данных. Работа направлена на выявление ключевых характеристик студентов, которые коррелируют с их учебными достижениями.

### Цели работы
1. Выявить демографические, социальные и академические факторы, влияющие на успеваемость студентов.
2. Проанализировать взаимосвязи между различными характеристиками студентов и их средним баллом.
3. Определить статистически значимые предикторы академической успеваемости.
4. Разработать рекомендации для образовательных учреждений по улучшению успеваемости студентов.

### Описание исходных данных
Исходный набор данных представляет собой анкеты студентов, содержащие 30 признаков для каждого студента. Файл взят из образовательного курса. Данные включают:

**Демографические характеристики:**
- Возраст, пол, рост, вес, размер обуви, длина волос
- Семейное положение, наличие братьев/сестер
- Расстояние до дома

**Академические показатели:**
- Оценки по 11 предметам (русский язык, математика, физика, информатика, химия, литература, история, география, биология, иностранный язык, обществознание)
- Номер курса, институт
- Наличие проблем в прошлом семестре

**Социально-бытовые характеристики:**
- Проживание в общежитии
- Количество друзей
- Время, проводимое в социальных сетях
- Пригодность к службе в армии

**Личные предпочтения:**
- Любимые животные
- Предпочтения в фастфуде и шоколаде
- Выбор в игре "камень-ножницы-бумага"
- Место в самолете

Общий объем данных: анкеты нескольких сотен студентов. Данные содержат пропущенные значения и требуют предварительной обработки.

### Полученные результаты

**1. Ключевые факторы успеваемости:**
- **Отрицательные корреляции**: количество изучаемых предметов (r ≈ -0.183), наличие проблем в прошлом семестре (r ≈ -0.267), мужской пол (r ≈ -0.128).
- **Положительные корреляции**: ношение очков (r ≈ 0.057), количество детей (r ≈ 0.079).
- **Незначимые факторы**: время в социальных сетях, расстояние до дома, количество друзей, рост, вес.

**2. Демографические закономерности:**
- Женщины показывают статистически значимо более высокую успеваемость по сравнению с мужчинами (p < 0.05).
- Успеваемость снижается на старших курсах.
- Студенты возрастной группы 20-21 год демонстрируют наивысшую успеваемость.

**3. Академические паттерны:**
- Обнаружены четкие кластеры коррелирующих предметов: естественнонаучный блок (математика, физика, информатика) и гуманитарный блок (литература, история, обществознание).
- Студенты, сильные в технических дисциплинах, часто показывают средние результаты в гуманитарных, и наоборот.

**4. Неочевидные зависимости:**
- Студенты, предпочитающие кошек, имеют несколько более высокую успеваемость.
- Выбор "бумаги" в игре "камень-ножницы-бумага" коррелирует с более высокой успеваемостью.
- Предпочтение определенных видов шоколада (KitKat, Twix) показывает слабую положительную связь с успеваемостью.

**5. Регрессионный анализ:**
Множественная линейная регрессия подтвердила значимость следующих предикторов:
- Количество изучаемых предметов (отрицательное влияние)
- Наличие проблем в прошлом семестре (отрицательное влияние)
- Пол (женский - положительное влияние)
- Курс обучения (отрицательное влияние)

### Заключение
Цель исследования достигнута: выявлены ключевые факторы, влияющие на успеваемость студентов. Наиболее значимыми предикторами оказались академические (количество предметов, наличие прошлых проблем) и демографические (пол, курс обучения) характеристики.

**Основные выводы:**
1. Учебная нагрузка (количество предметов) требует оптимизации, так как ее увеличение коррелирует со снижением успеваемости.
2. Студенты старших курсов нуждаются в дополнительной академической поддержке.
3. Женщины демонстрируют более высокую академическую дисциплину.

**Рекомендации для образовательных учреждений:**
1. Внедрить мониторинг успеваемости на старших курсах.
2. Разработать программы академической поддержки для студентов, имевших проблемы в прошлых семестрах.
3. Оптимизировать учебную нагрузку по количеству параллельно изучаемых дисциплин.
4. Использовать дифференцированный подход к обучению студентов разных институтов и специальностей.
